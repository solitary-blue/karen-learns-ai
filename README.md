# Karen Learns AI: Agentic Skills for Montessori Educators

This project is dedicated to helping Karen, a Montessori elementary school teacher, develop agentic AI skills for her upcoming master's program and her daily classroom work.
She will be assisted on her journey by lastobelus, a web developer with some experience in agentic programming & AI.

## Project Vision
To bridge the gap between Montessori philosophy and modern AI, empowering Karen to use AI agents as collaborative partners in education.

## Learning Roadmap

### Phase 1: AI as a Collaborative Partner (Prompting & Personas)
*   **Goal:** Understand how to "prime" an AI with Montessori principles.
*   **Exercise 1:** Creating the "Montessori Expert" persona.

### Phase 2: Introduction to Agents (Autonomy & Tools)
*   **Goal:** Learn what makes an AI an "agent" (reasoning + tools).

### Phase 3: Building the Montessori Assistant (Hands-on Prototype)
*   **Goal:** Build and modify a simple agentic tool.
